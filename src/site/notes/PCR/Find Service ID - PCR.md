# Find Service ID
1. Copy the Jack ID
	1. (`MSC-2E-02-012A`)
2. Paste it into `Inventory : Equipment`
3. If no rows appear
	1. Copy the Switch ID
	2. Paste it into `Inventory : Equipment`
	3. Click on the first row that appears
	4. Click on `Port/Pin/Lens`
	5. Type slash then the port number that comes after the slash `/##`
	6. If there is a number in the `Service` cell
		1. Right click on cell in `Service` 
		2. Click `Copy Cell`
		3. This is the `Service ID`, you are done
	7. If the `Service` cell is empty
		1. If you are doing a Jack Deactivate you can stop as it was never activated in the first place
		2. If you are doing a jack move, then you need to add it and idk how yet :)
4. If a row does appear, continue
5. Click on `Port/Pins/Lens`
6. Look at the `Service` cell
7. If there is a number in the `Service` cell
	1. Right click on cell in `Service` 
	2. Click `Copy Cell`
	3. This is the `Service ID`, you are done
8. If the `Service` cell is empty
	1. If you are doing a Jack Deactivate you can stop as it was never activated in the first place
	2. If you are doing a jack move, then you need to add it and idk how yet :)
