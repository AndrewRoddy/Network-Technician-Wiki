# Finish Service Order
1. Switch `Status:` to `Complete`
2. Then press the stuff that says it is done
3. In `Closuer Comments` type
```
PRC Completed and Finalized
```
4. Switch the `Status` to `Resolved` 
5. Yay another PCR done!
