# Jack Move
Spreadsheet : `data` `Floor` `Room` `Aruba SWX` `Aruba Port` `Jack` `Cisco Port`

1. Add action
2. Change `Action` to `Move`
3. Paste the `data` ID to the `Service` area
4. If it has a room number add it in the room number place
5. Then press save
6. Press `Cabling`
7. Change the switch
8. In the `Origination Equipment` paste in the new `Aruba SWX` switch.
9. All you have to change is the cabling. All you need to change is the in and the out.
