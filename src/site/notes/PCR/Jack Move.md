# Jack Move
Spreadsheet rules : Green -> `Move`, Yellow -> `Unpatch`
Spreadsheet : `data` `Floor` `Room` `Aruba SWX` `Aruba Port` `Jack` `Cisco Port`

1. Press `Add`
2. Change `Action` to `Move`
3. Paste the spreadsheet `data` ID number into the `Service:` search bar
4. If text in the `Room` cell exists 
	1. In `Move To Location:` type in the building name (`CPA`) 
	2. Then add a space and type in the room number (`D101B`) 
5. Then press `Save New` 
6. Click check box in the top left
7. Press `Save Without SDC`
8. Press `Cabling` tab
9. Double click on `In` Action 
10. Copy `Aruba SWX ` and paste it in `Origination Equipment:` 
11. Copy `Aruba Port` and paste it in `Origination Prt/Pin/Len:` 
12. Press `Save and Close`
13. Press `Save and Close` again
